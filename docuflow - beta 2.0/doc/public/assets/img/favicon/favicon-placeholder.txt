FAVICON FILES NEEDED:
====================

1. favicon.ico (32x32 pixels)
   - Classic icon format for old browsers
   - Can contain multiple sizes in one file
   - Required for Internet Explorer compatibility

2. favicon-32x32.png (32x32 pixels)
   - Modern standard for most browsers
   - PNG format with transparency support
   - Used in browser tabs and bookmarks

3. favicon-16x16.png (16x16 pixels)
   - Small size for taskbar and favorites
   - Prevents blurry scaling from larger icons
   - Important for Windows taskbar display

4. apple-touch-icon.png (180x180 pixels)
   - iOS devices (iPhone, iPad, iPod)
   - Used when user adds site to home screen
   - Apple automatically adds rounded corners and gloss

HOW TO CREATE:
==============
1. Design your logo in a square format
2. Export in these exact sizes
3. Use online tools like favicon.io or realfavicongenerator.net
4. Place all files in this directory

TEMPORARY SOLUTION:
==================
You can use any square image and rename it to favicon.ico
as a temporary solution until proper icons are created.
